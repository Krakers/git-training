What exactly is git?

git is a version control system created by Linus Torvalds:

What's a version control system anyway?

  • Version Control Systems (VCS for short) are programs that keep track of changes in our files
  • Because of them, multiple people can work on the same files, without colliding with one another
  • They don't have to be source code files, designers use git too!
Why do we need it?

We need git for following reasons:

  • Since we have a lot of files in our project, we need to make sure that everyone has exactly the same version
  • We need to ensure that multiple people can work on a single file (like in automated tests)
  • We need a system that handles all file changes for us, using email, FTP or pigeon mail is our of the question
